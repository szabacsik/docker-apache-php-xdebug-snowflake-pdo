# docker-apache-php-xdebug-snowflake-pdo
Docker Apache PHP XDebug Snowflake PDO

### Spin up the container
```shell
cd docker
make up
```

### Get a bash shell in the container
```shell
cd docker
make bash
```

### Stop the container
```shell
cd docker
make down
```

### PHPInfo
<http://127.0.0.1:8080/info.php>

### PHP PDO driver for Snowflake
<https://github.com/snowflakedb/pdo_snowflake>
